常见问题

洛谷题目数据生成器支持多种常见数据结构，包括但不限于：数组、链表、栈、队列、树（二叉树、B树、AVL树等）、图（有向图、无向图、带权图等）、字符串、哈希表等。同时，您还可以通过自定义脚本来支持更复杂的数据结构。

是的，生成的数据可以直接用于洛谷提交。洛谷题目数据生成器支持生成符合洛谷格式要求的输入输出文件，您可以将生成的文件打包上传到洛谷平台作为题目的测试数据。

洛谷题目数据生成器支持使用 Python 编写自定义脚本。您可以通过界面中的"自定义脚本"功能，编写自己的数据生成逻辑。系统提供了丰富的 API 和工具函数，帮助您快速生成各类测试数据。详细的脚本编写指南请参考用户手册中的"自定义脚本"章节。

如果在生成大规模数据时遇到内存不足的问题，您可以尝试以下方法：1) 分批次生成数据，然后合并结果；2) 使用生成器模式编写脚本，避免一次性加载所有数据；3) 增加系统内存或使用内存优化选项；4) 对于特别大的数据，可以考虑使用外部排序等技术。
